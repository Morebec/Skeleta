Box

Most layouts need some form of container that allows specifying zones of content with potential padding and sizes.

To fullfill this need, the box is a simple component that sets the display property of an element to block and that supports various modifiers for padding, width and height.

  • padding: box--paddingN
  • width: box--widthN, box--min-widthN, box--max-widthN
  • height: base, min, max

This component only handles the layout related properties of a box and delegates the actual design to the user.

Example

Here's an example of a card component based on a box:
